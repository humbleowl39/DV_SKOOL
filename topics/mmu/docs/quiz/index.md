# MMU 퀴즈

각 챕터의 핵심 개념 이해도를 확인합니다. 단순 암기로는 답하기 어렵도록 Bloom's Taxonomy 분포를 적용했습니다. 문항에 답하면서 "왜 이 답인지" 설명할 수 있다면 개념을 진짜로 이해한 것입니다. Bloom's Taxonomy 분포에 따라 **암기(Remember)/이해(Understand)/적용(Apply)/분석(Analyze)/평가(Evaluate)** 수준의 문항이 섞여 있습니다.

## 사용법

1. 챕터 본문을 학습한 후 해당 모듈의 퀴즈 페이지로 이동
2. 정답을 머릿속으로 정한 뒤 **"정답 / 해설"** 영역을 펼쳐 확인
3. 틀린 문항은 본문 해당 절을 다시 학습
