## 1. Why care? вҖ” мқҙ лӘЁл“Ҳмқҙ мҷң н•„мҡ”н•ңк°Җ

мқҙнӣ„ лӘЁл“  ARM Security лӘЁл“ҲмқҖ н•ң к°Җм •м—җм„ң м¶ңл°ңн•©лӢҲлӢӨ вҖ” **"к°ҷмқҖ SoC м•Ҳм—җм„ң л‘җ мў…лҘҳмқҳ SW к°Җ, м„ңлЎң л©”лӘЁлҰ¬В·л Ҳм§ҖмҠӨн„°В·мқён„°лҹҪнҠёлҘј ліҙм§Җ лӘ»н•ң мұ„ кіөмЎҙн•ңлӢӨ"**. SCR_EL3.NS к°Җ мҷң EL3 м—җм„ңл§Ң л°”лҖҢлҠ”м§Җ, BootROM мқҙ мҷң EL3 м—җм„ң мӢңмһ‘н•ҳлҠ”м§Җ, TZASC/TZPC/SMMU/GIC к°Җ мҷң NS bit лҘј л”°лқјк°ҖлҠ”м§Җ вҖ” м „л¶Җ мқҙ н•ң к°Җм •мқҳ нҢҢмғқмһ…лӢҲлӢӨ.

мқҙ лӘЁл“Ҳмқ„ кұҙл„Ҳлӣ°л©ҙ мқҙнӣ„мқҳ лӘЁл“  spec/л Ҳм§ҖмҠӨн„°/кІҖмҰқ кІ°м •мқҙ "к·ёлғҘ мҷёмӣҢм•ј н•ҳлҠ” к·ңм№ҷ" мңјлЎң ліҙмһ…лӢҲлӢӨ. л°ҳлҢҖлЎң EL Г— NS лқјлҠ” **л‘җ м¶•мқҳ л§ӨнҠёлҰӯмҠӨ** лҘј м •нҷ•нһҲ мһЎкі  лӮҳл©ҙ, мғҲ register лҘј л§ҢлӮ  л•Ңл§ҲлӢӨ _"мқҙкұҙ м–ҙлҠҗ м№ёмқҳ ліҙнҳё мһҗмӣҗмқёк°Җ"_ мІҳлҹј мң„м№ҳк°Җ ліҙмһ…лӢҲлӢӨ.

## 2. Intuition вҖ” л№„мң мҷҖ н•ң мһҘ к·ёлҰј

!!! tip "рҹ’Ў н•ң мӨ„ л№„мң "
    **Exception Level + TrustZone** вүҲ _кұҙл¬јмқҳ к¶Ңн•ң л“ұкёү (EL) Г— ліҙм•Ҳ/мқјл°ҳ м¶ңмһ…мҰқ (NS bit)_.<br>
    EL мқҖ **мёө** (B1 lobby = EL0, 1F = EL1, ... мҳҘмғҒ = EL3), NS bit лҠ” **м¶ңмһ…мҰқ мғүк№”** (нҢҢлһҖмғү = Non-Secure, л№Ёк°„мғү = Secure). к°ҷмқҖ мёөмқҙлқјлҸ„ м¶ңмһ…мҰқ мғүмқҙ лӢӨлҘҙл©ҙ лӢӨлҘё л°©. мҳҘмғҒ (EL3) мқҖ н•ӯмғҒ л№Ёк°„ м¶ңмһ…мҰқ м „мҡ© вҖ” кұ°кё°м„ңл§Ң NS bit лҘј лӢӨлҘё мғүмңјлЎң л°”кҝ”мЈјлҠ” кІҢмқҙнҠёк°Җ м—ҙлҰҪлӢҲлӢӨ.

### мҷң мқҙл ҮкІҢ м„Өкі„лҗҗлҠ”к°Җ вҖ” Design rationale

м„ё к°Җм§Җ мҡ”кө¬к°Җ лҸҷмӢңм—җ н’Җл Өм•ј н–ҲмҠөлӢҲлӢӨ.

1. **OS к°Җ н•ҙнӮ№лҸјлҸ„ кІ°м ң нӮӨ/мғқмІҙ лҚ°мқҙн„°лҠ” мӮҙм•„ мһҲм–ҙм•ј н•ңлӢӨ** вҶ’ л‘җ к°ңмқҳ кІ©лҰ¬лҗң мӢӨн–ү нҷҳкІҪ (S/NS World).
2. **кІ©лҰ¬к°Җ SW мӢ лў° (мң„ліҖмЎ°лҗң OS к°Җ мһҗкё° к¶Ңн•ңмқ„ м •м§Ғн•ҳкІҢ ліҙкі н•ҳкёё кё°лҢҖ) к°Җ м•„лӢҲлқј HW к°•м ңм—¬м•ј н•ңлӢӨ** вҶ’ лӘЁл“  лІ„мҠӨ transaction м—җ **NS bit** к°Җ н•Ёк»ҳ нқҗлҘҙкі , downstream filter к°Җ кұ°л¶Җ.
3. **к·ёлҹ¬лӮҳ л‘ҳмқҙ к°ҖлҒ”мқҖ нҶөмӢ н•ҙм•ј н•ңлӢӨ** (м•ұмқҙ кІ°м ң мҡ”мІӯ вҶ’ TEE мқ‘лӢө) вҶ’ **лӢЁмқј кІҢмқҙнҠё (EL3)** л§Ң NS bit лҘј л°”кҝҖ мҲҳ мһҲкІҢ н•ҳкі , к·ё кІҢмқҙнҠёмқҳ мҪ”л“ң (BL31) лҘј к°ҖмһҘ мһ‘кІҢ кІҖмҰқ.

мқҙ м„ё мҡ”кө¬мқҳ көҗм§‘н•©мқҙ кі§ EL Г— NS л§ӨнҠёлҰӯмҠӨ + EL3 лӢЁмқј кІҢмқҙнҠё + NS bit мқҳ HW м „нҢҢмһ…лӢҲлӢӨ. мқҙ л””мһҗмқёмқҖ мқҙнӣ„ SCR_EL3, VBAR_ELn, TZPC/TZASC, GIC group, cache NS tag вҖ” мҰү ARM Security мқҳ кұ°мқҳ лӘЁл“  м–ҙнңҳлҘј кІ°м •н•©лӢҲлӢӨ.

```c
// Step в‘Ў мқҳ NS-EL1 kernel мёЎ н•өмӢ¬ вҖ” мқҙ н•ң лІҲмқҳ SMC к°Җ в‘ў~в‘Ё лҘј нҠёлҰ¬кұ°
register uint64_t x0 asm("x0") = SMC_FID_OPTEE_CALL_WITH_ARG;
register uint64_t x1 asm("x1") = (uintptr_t)&optee_msg_arg;
asm volatile("smc #0" : "+r"(x0), "+r"(x1));
/* x0 returns: OPTEE_SMC_RETURN_OK / RPC / etc. */
```

!!! note "м—¬кё°м„ң мһЎм•„м•ј н•  л‘җ к°Җм§Җ"
    **(1) мӣ”л“ң м „нҷҳмқҖ EL3 к°•м ң вҖ” NS-EL1 вҶ’ S-EL1 м§Ғн–ү кІҪлЎңлҠ” м•„нӮӨн…ҚмІҳм ҒмңјлЎң мЎҙмһ¬н•ҳм§Җ м•ҠмҠөлӢҲлӢӨ.** SMC trap вҶ’ BL31 вҶ’ ERET мқҙ мң мқјн•ң нҶөлЎң. <br>
    **(2) к¶Ңн•ң мғҒмҠ№ (EL0 вҶ’ EL1 вҶ’ EL3) мқҖ н•ӯмғҒ Exception мңјлЎңл§Ң л°ңмғқ** вҖ” SW к°Җ мһ„мқҳлЎң PSTATE.EL мқ„ мҳ¬л Ө м“ё мҲҳ м—ҶмҠөлӢҲлӢӨ. мқҙкІҢ ліҙм•Ҳ лӘЁлҚёмқҳ лјҲлҢҖ.

## 4. мқјл°ҳнҷ” вҖ” л‘җ м¶• (мҲҳм§Ғ EL, мҲҳнҸү NS) кіј 8 mode л§ӨнҠёлҰӯмҠӨ

### 4.1 л‘җ м¶•мқҳ м •мқҳ

!!! note "м •мқҳ (ISO 11179)"
    - **Exception Level (EL)**: ARMv8 PE (Processing Element) мқҳ к¶Ңн•ң кі„мёө (`EL0` < `EL1` < `EL2` < `EL3`) мңјлЎң, к°Ғ level мқҙ system register / л©”лӘЁлҰ¬ л§Өн•‘ / лӘ…л №м–ҙ мӮ¬мҡ© к¶Ңн•ң лІ”мң„лҘј кІ°м •н•ңлӢӨ.
    - **Security State (NS bit)**: PE мқҳ 1-bit мғҒнғңлЎң, нҳ„мһ¬ instruction мқҙ Secure World м—җм„ң мӢӨн–үлҗҳлҠ”м§Җ (`NS=0`) Non-Secure World м—җм„ң мӢӨн–үлҗҳлҠ”м§Җ (`NS=1`) лҘј н‘ңмӢңн•ҳл©°, лӘЁл“  outgoing bus transaction мқҳ ліҙм•Ҳ attribute лЎң м „нҢҢлҗңлӢӨ.

```
    EL (мҲҳм§Ғ, к¶Ңн•ң)               NS (мҲҳнҸү, мӣ”л“ң)
   в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ             в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
   EL3 в”Җ Secure Monitor          NS=0 в”Җ Secure   (TEE)
   EL2 в”Җ Hypervisor              NS=1 в”Җ Non-Secure (Rich OS)
   EL1 в”Җ OS kernel
   EL0 в”Җ Application
```

н•өмӢ¬: л‘җ м¶•мқҖ **лҸ…лҰҪ** мһ…лӢҲлӢӨ. EL3 л§Ң н•ӯмғҒ NS=0 мңјлЎң кі м •лҗҳкі  (м „нҷҳ кІҢмқҙнҠё), лӮҳлЁём§Җ EL мқҖ (S, NS) л‘ҳ лӢӨ к°ҖлҠҘ. к·ёлһҳм„ң н‘ңл©ҙмғҒ 4 Г— 2 = 8 mode к°Җ мЎҙмһ¬н•©лӢҲлӢӨ.


- **EL3 лҠ” н•ӯмғҒ Secure** вҖ” NS=1 EL3 лқјлҠ” mode лҠ” м•„нӮӨн…ҚмІҳм ҒмңјлЎң м •мқҳлҗҳм§Җ м•ҠмҠөлӢҲлӢӨ (к·ёлһҳм„ң 8 - 1 = 7 mode к°Җ мӢӨмҡ©м ҒмңјлЎң мЎҙмһ¬).

### 5.1 Exception Level (EL0 ~ EL3)

```
к¶Ңн•ң лҶ’мқҢ

  |

  |      - мөңкі  к¶Ңн•ң, Secure/Non-Secure м „нҷҳ кҙҖлҰ¬
```

### 5.2 TrustZone вҖ” Secure / Non-Secure л¶„лҰ¬

#### мҷң л‘җ к°ңмқҳ "мӣ”л“ң" к°Җ н•„мҡ”н•ңк°Җ?

```
л¬ём ң: мқјл°ҳ OSк°Җ н•ҙнӮ№лҗҳл©ҙ?
  вҶ’ OS м»Өл„җ к¶Ңн•ң нғҲм·Ё вҶ’ лӘЁл“  л©”лӘЁлҰ¬/л””л°”мқҙмҠӨ м ‘к·ј к°ҖлҠҘ
  вҶ’ кІ°м ң м •ліҙ, м•”нҳё нӮӨ, мғқмІҙ лҚ°мқҙн„° л…ём¶ң

TrustZone н•ҙкІ°:
  Secure World (TEE)          Normal World
  +-------------------+      +-------------------+
  | кІ°м ң мІҳлҰ¬          |      | мқјл°ҳ м•ұ           |
  | м•”нҳё нӮӨ м ҖмһҘ       |      | лёҢлқјмҡ°м Җ          |
  | мғқмІҙ мқёмҰқ          |      | кІҢмһ„              |
  | DRM ліөнҳёнҷ”         |      | OS (Linux)        |
  +-------------------+      +-------------------+
         вҶ‘ HW кІ©лҰ¬                  вҶ‘
         |                          |
  Normal Worldм—җм„ң Secure World л©”лӘЁлҰ¬ м ‘к·ј л¶Ҳк°Җ (HW к°•м ң)
  вҶ’ OSк°Җ н•ҙнӮ№лҗҳм–ҙлҸ„ Secure Worldмқҳ нӮӨ/лҚ°мқҙн„°лҠ” м•Ҳм „
```

#### TrustZone мқҳ HW кІ©лҰ¬ л©”м»ӨлӢҲмҰҳ

```
лӘЁл“  лІ„мҠӨ нҠёлһңмһӯм…ҳм—җ NS (Non-Secure) л№„нҠё м¶”к°Җ:

  +--------+    +------+-------+------+
  | Master | вҶ’  | NS=0 | ADDR  | DATA |  Secure м ‘к·ј
  +--------+    +------+-------+------+
                | NS=1 | ADDR  | DATA |  Non-Secure м ‘к·ј
                +------+-------+------+

  NS=0 (Secure): Secure л©”лӘЁлҰ¬/л””л°”мқҙмҠӨ м ‘к·ј к°ҖлҠҘ
  NS=1 (Non-Secure): Secure мҳҒм—ӯ м ‘к·ј мӢң вҶ’ лІ„мҠӨ м—җлҹ¬ (м°ЁлӢЁ)

  вҶ’ NS л№„нҠёлҠ” HWк°Җ к°•м ң вҖ” SWлЎң мЎ°мһ‘ л¶Ҳк°ҖлҠҘ
  вҶ’ EL3 (Secure Monitor)л§Ң NS л№„нҠёлҘј ліҖкІҪн•  мҲҳ мһҲмқҢ
```

### 5.3 SCR_EL3 (Secure Configuration Register)

```
EL3м—җм„ң м ңм–ҙн•ҳлҠ” н•өмӢ¬ ліҙм•Ҳ л Ҳм§ҖмҠӨн„°:

  SCR_EL3.NS  (bit[0]): 0=Secure, 1=Non-Secure
    вҶ’ EL3к°Җ н•ҳмң„ ELмқҳ ліҙм•Ҳ мғҒнғңлҘј кІ°м •

  SCR_EL3.IRQ (bit[1]): IRQлҘј EL3лЎң лқјмҡ°нҢ…
  SCR_EL3.FIQ (bit[2]): FIQлҘј EL3лЎң лқјмҡ°нҢ…
  SCR_EL3.SMD (bit[7]): SMC лӘ…л № л№„нҷңм„ұнҷ”
  SCR_EL3.HCE (bit[8]): EL2 нҷңм„ұнҷ”
  SCR_EL3.RW  (bit[10]): н•ҳмң„ ELмқҳ AArch64/32 м„ нғқ

  BootROM (EL3)мқҙ SCR_EL3лҘј м„Өм •н•ҳм—¬ ліҙм•Ҳ м •мұ…мқ„ кІ°м •
  вҶ’ BL2лЎң м җн”„н•  л•Ң NS=0 (Secure) мң м§Җ
  вҶ’ BL33мңјлЎң м җн”„н•  л•Ң NS=1 (Non-Secure) м „нҷҳ
```

### 5.4 EL м „нҷҳ лӘ…л №м–ҙ мҡ”м•Ҫ

```
мғҒн–Ҙ м „нҷҳ (Lower EL вҶ’ Higher EL): Exception л°ңмғқ
  EL0 вҶ’ EL1:  SVC (Supervisor Call)    вҖ” мӢңмҠӨн…ң мҪң
  EL1 вҶ’ EL2:  HVC (Hypervisor Call)    вҖ” Hypervisor м„ңл№„мҠӨ мҡ”мІӯ
  Any вҶ’ EL3:  SMC (Secure Monitor Call) вҖ” ліҙм•Ҳ м„ңл№„мҠӨ / мӣ”л“ң м „нҷҳ

  к·ё мҷё Exception:
    IRQ/FIQ:  GIC м„Өм •м—җ л”°лқј EL1/EL2/EL3лЎң лқјмҡ°нҢ…
    Data Abort, Instruction Abort:  нҳ„мһ¬ EL лҳҗлҠ” мғҒмң„ EL
    SError:  л№„лҸҷкё° м—җлҹ¬ вҶ’ м„Өм •м—җ л”°лқј EL3к№Ңм§Җ к°ҖлҠҘ

н•ҳн–Ҙ м „нҷҳ (Higher EL вҶ’ Lower EL): ERET лӘ…л №
  ERET:  Exception Return
    вҶ’ SPSR_ELnм—җм„ң ліөк·Җн•  ELкіј PSTATE ліөмӣҗ
    вҶ’ ELR_ELnм—җм„ң ліөк·Җ мЈјмҶҢ(PC) ліөмӣҗ
    вҶ’ н•ӯмғҒ к°ҷмқҖ ELмқҙкұ°лӮҳ лҚ” лӮ®мқҖ ELлЎңл§Ң ліөк·Җ к°ҖлҠҘ

мЈјмқҳ: SWк°Җ мһ„мқҳлЎң ELмқ„ мҳ¬лҰҙ мҲҳ м—ҶмқҢ вҖ” л°ҳл“ңмӢң Exception кІҪмң 
      вҶ’ мқҙкІғмқҙ ліҙм•Ҳмқҳ н•өмӢ¬: к¶Ңн•ң мғҒмҠ№мқҖ HWк°Җ нҶөм ң
```

    **мҷң н—·к°ҲлҰ¬лҠ”к°Җ**: "кё°лҠҘ мјңм§җ = м•Ҳм „" мқҙлқјлҠ” м§ҒкҙҖ. мӢӨм ң ліҙм•ҲмқҖ boot лӢЁкі„м—җм„ң secure-only мһҗмӣҗмқҳ lock-down м •нҷ•м„ұм—җ мқҳмЎҙ.

!!! danger "вқ“ мҳӨн•ҙ 2 вҖ” 'EL мқҖ SW к°Җ мһҗмң лЎӯкІҢ мҳ¬лҰҙ мҲҳ мһҲлӢӨ'"
    **мӢӨм ң**: к¶Ңн•ң мғҒмҠ№мқҖ **л°ҳл“ңмӢң Exception мңјлЎңл§Ң** к°ҖлҠҘн•©лӢҲлӢӨ. SW к°Җ PSTATE.EL мқ„ м§Ғм ‘ м“°л Өкі  н•ҳл©ҙ UNDEFINED лҳҗлҠ” л¬ҙмӢң. SVC/HVC/SMC/IRQ/FIQ/Abort л§Ңмқҙ мғҒн–Ҙ нҠёлҰ¬кұ°мқҙкі , н•ҳн–ҘмқҖ ERET л§Ңмқҙ к°ҖлҠҘ. мқҙ л№„лҢҖм№ӯмқҙ ліҙм•Ҳ лӘЁлҚёмқҳ ліём§Ҳмһ…лӢҲлӢӨ.<br>
    **мҷң н—·к°ҲлҰ¬лҠ”к°Җ**: мқјл°ҳ ISA мқҳ "mode bit лҘј set" к°ҷмқҖ м§ҒкҙҖ л•Ңл¬ём—җ.

!!! danger "вқ“ мҳӨн•ҙ 3 вҖ” 'NS-EL1 вҶ’ S-EL1 м§Ғн–ү к°ҖлҠҘ'"
    **мӢӨм ң**: л‘җ EL мӮ¬мқҙм—җ м§Ғм ‘ м „мқҙ лӘ…л №мқҖ м—ҶмҠөлӢҲлӢӨ. л°ҳл“ңмӢң EL3 (SMC вҶ’ BL31 вҶ’ ERET) кІҪмң . мқҙкІҢ лӢЁмқј кІҢмқҙнҠё л””мһҗмқёмқҳ н•өмӢ¬.<br>
    **мҷң н—·к°ҲлҰ¬лҠ”к°Җ**: "к°ҷмқҖ EL л ҲлІЁмқҙлӢҲ мҳҶмңјлЎңл§Ң к°Җл©ҙ лҗ  кІғ к°ҷлӢӨ" лҠ” м¶”лЎ .

!!! danger "вқ“ мҳӨн•ҙ 4 вҖ” 'EL3 к°Җ н•ӯмғҒ нҷңм„ұнҷ”лҗҳм–ҙ мһҲлӢӨ'"
    **мӢӨм ң**: мқјл¶Җ SoC (м Җк°Җнҳ• Cortex-A лҳҗлҠ” R-class) лҠ” EL3 лҘј implement н•ҳм§Җ м•ҠмҠөлӢҲлӢӨ (`ID_AA64PFR0_EL1.EL3 = 0`). к·ё кІҪмҡ° secure вҶ” non-secure м „нҷҳ мһҗмІҙк°Җ м—Ҷкұ°лӮҳ EL2 к°Җ monitor м—ӯн• мқ„ лҢҖмӢ н•©лӢҲлӢӨ. кІҖмҰқ мӢң мҡ°лҰ¬ SoC мқҳ EL3 implementation м—¬л¶ҖлҘј лЁјм Җ нҷ•мқё.<br>
    **мҷң н—·к°ҲлҰ¬лҠ”к°Җ**: н‘ңмӨҖ spec мқҳ "EL3 к°Җ monitor" н‘ңнҳ„ л•Ңл¬ём—җ.

!!! danger "вқ“ мҳӨн•ҙ 5 вҖ” 'NS bit лҠ” SW к°Җ мһ„мқҳлЎң set к°ҖлҠҘ'"
    **мӢӨм ң**: NS bit лҠ” SCR_EL3.NS к°Җ source-of-truth мқҙкі , мҳӨм§Ғ EL3 м—җм„ңл§Ң write к°ҖлҠҘн•©лӢҲлӢӨ. NS-EL1 мқҙлӮҳ EL2 м—җм„ң SCR_EL3 м—җ write мӢңлҸ„ вҶ’ trap (лҳҗлҠ” silently ignored). к·ёлһҳм„ң мқҙ bit к°Җ HW кІ©лҰ¬мқҳ м¶ңл°ңм җ.<br>
    **мҷң н—·к°ҲлҰ¬лҠ”к°Җ**: "к·ёлғҘ register 1 bit" лқјлҠ” мқёмғҒ.

### DV л””лІ„к·ё мІҙнҒ¬лҰ¬мҠӨнҠё (мқҙ лӘЁл“Ҳ лӮҙмҡ©мңјлЎң л§ҲмЈјм№  мІ« мӢӨнҢЁл“Ө)

| мҰқмғҒ | 1м°Ё мқҳмӢ¬ | м–ҙл”” ліҙлӮҳ |
|---|---|---|
| NS world к°Җ secure DRAM read н–ҲлҠ”лҚ° BusError к°Җ м•Ҳ лӮҳмҳҙ | SCR_EL3.NS bit к°Җ outgoing transaction attribute (`AxPROT[1]`/`AxNSE`) лЎң м „нҢҢ м•Ҳ лҗЁ | master IF мқҳ NS attribute мҷҖ SCR_EL3.NS мқҳ cycle-by-cycle мқјм№ҳ SVA |
| SMC нӣ„ secure GPR мһ”м—¬ к°’мқҙ NS world м—җ ліҙмһ„ | BL31 мқҳ context save/restore к°Җ NEON/FP/SVE лҲ„лқҪ | BL31 context мҪ”л“ң + post-ERET register dump |
| `SVC` к°Җ EL3 к№Ңм§Җ мҳ¬лқјк°җ | trap routing (`HCR_EL2.IMO/AMO`, `SCR_EL3.IRQ/FIQ`) мһҳлӘ» | exception entry мқҳ VBAR base мҷҖ offset 0x000 vs 0x400 |
| EL2 м—җм„ң ERET н–ҲлҠ”лҚ° EL3 лЎң м җн”„ | SPSR_EL2.M мһҳлӘ» (EL3h лЎң set лҗЁ) | SPSR.M л””мҪ”л“ң + ERET м§Ғм „ dump |
| Stage2 к°Җ л№„нҷңм„ұмқёлҚ° VM кІ©лҰ¬ мӢӨнҢЁ | VTTBR_EL2/VTCR_EL2 лҜём„Өм • вҶ’ identity mapping | VTCR_EL2.T0SZ/SL0 мҷҖ VTTBR_EL2 к°Җ valid S2 table к°ҖлҰ¬нӮӨлҠ”м§Җ |
| OTP register к°Җ NS world м—җм„ң мқҪнһҳ | TZPC slave л¶„лҘҳ лҲ„лқҪ | BL1 boot trace + TZPC region м„Өм • |
| к°ҷмқҖ PA мқёлҚ° S/NS к°Җ к°ҷмқҖ мәҗмӢң лқјмқё | cache NS tag к°Җ л№„нҷңм„ұ лҳҗлҠ” line invalidate лҲ„лқҪ | cache controller мқҳ NS tag bit + write-back м •мұ… |
| ERET нӣ„ IRQ mask н’ҖлҰј | SPSR.DAIF мһҳлӘ» ліөмӣҗ | ERET м§Ғм „ SPSR мқҳ DAIF field |

мқҙ мІҙнҒ¬лҰ¬мҠӨнҠёлҠ” мқҙнӣ„ лӘЁл“Ҳм—җм„ң лҚ” м •көҗн•ң нҳ•нғңлЎң лӢӨмӢң лӮҳмҳөлӢҲлӢӨ. м§ҖкёҲ лӢЁкі„м—җм„ңлҠ” "ARM Security мӢӨнҢЁ = (SCR_EL3 | SPSR | VBAR | TZPC | TZASC | NS attribute) мӨ‘ н•ҳлӮҳ" л§Ң кё°м–өн•ҳм„ёмҡ”.

---

!!! warning "мӢӨл¬ҙ мЈјмқҳм җ вҖ” SCR_EL3.NS л№„нҠё м „нҢҢ лҲ„лқҪ"
    **нҳ„мғҒ**: NS world м—җм„ң secure-only л©”лӘЁлҰ¬/л Ҳм§ҖмҠӨн„°лҘј read н–ҲлҠ”лҚ° BusError м—Ҷмқҙ м •мғҒ лҚ°мқҙн„°к°Җ л°ҳнҷҳлҗңлӢӨ.

    **мӣҗмқё**: EL3 м§„мһ…/нғҲм¶ң мӢң SCR_EL3.NS л№„нҠёк°Җ нҠёлһңмһӯм…ҳ м–ҙнҠёлҰ¬л·°нҠёлЎң лІ„мҠӨм—җ м •нҷ•нһҲ м „нҢҢлҗҳм§Җ м•Ҡм•„, downstream filter (TZASC/TZPC) к°Җ NS нҠёлһңмһӯм…ҳмқ„ secure лЎң мҳӨмқён•ңлӢӨ.

    **м җкІҖ нҸ¬мқёнҠё**: AxPROT[1]/AxNSE л“ұ NS м–ҙнҠёлҰ¬л·°нҠёк°Җ лӘЁл“  master IF м—җм„ң SCR_EL3.NS мҷҖ мқјм№ҳн•ҳлҠ”м§Җ SVA лЎң к°•м ңн•ҳкі , NS world coverage мӢң secure resource read к°Җ м°ЁлӢЁлҗҳлҠ”м§Җ нҷ•мқё.

## 7. н•өмӢ¬ м •лҰ¬ (Key Takeaways)

- **л‘җ м¶•мқҳ ліҙм•Ҳ**: мҲҳм§Ғ (Exception Level: EL0 user вҶ’ EL1 kernel вҶ’ EL2 hypervisor вҶ’ EL3 secure monitor) + мҲҳнҸү (Secure World vs Non-Secure World).
- **EL3 = лӢЁмқј кІҢмқҙнҠё**: к°ҖмһҘ privileged + н•ӯмғҒ Secure + мӣ”л“ң м „нҷҳмқҳ мң мқјн•ң нҶөлЎң. ARM Trusted Firmware BL31 мқҙ м—¬кё°м„ң лҸҷмһ‘.
- **NS bit**: PSTATE мқҳ 1-bit мғҒнғңмқҙмһҗ лӘЁл“  outgoing bus transaction мқҳ attribute. EL3 л§Ң ліҖкІҪ к°ҖлҠҘ, HW к°Җ лӘЁл“  downstream м—җ к°•м ң м „нҢҢ.
- **к¶Ңн•ң мғҒмҠ№мқҖ Exception мңјлЎңл§Ң**: SVC/HVC/SMC/IRQ вҖ” SW к°Җ PSTATE.EL мқ„ мһ„мқҳлЎң set л¶Ҳк°Җ. н•ҳн–ҘмқҖ ERET мңјлЎңл§Ң.
- **8 mode л§ӨнҠёлҰӯмҠӨ (мӢӨмҡ© 7 mode)**: EL0~3 Г— S/NS, лӢЁ EL3 лҠ” н•ӯмғҒ Secure. кІҖмҰқ мӢң к°Ғ mode м—җм„ң м–ҙл–Ө мһҗмӣҗмқҙ visible/accessible н•ңм§Җ к·ёлҰјмңјлЎң к·ёл Өл‘җл©ҙ л¶„м„қмқҙ л№ лҰ…лӢҲлӢӨ.

!!! warning "мӢӨл¬ҙ мЈјмқҳм җ"
    - "TrustZone мјңм§җ = м•Ҳм „" мқҙ м•„лӢҲлқј _TZASC/TZPC/GIC/SMMU мқҳ boot-time м„Өм • м •нҷ•м„ұ_ мқҙ кі§ м•Ҳм „мһ…лӢҲлӢӨ.
    - SCR_EL3.NS ліҖкІҪмқҖ EL3 мқҳ _лӘ…мӢңм Ғ write_ мӢңм җл§Ң мқјм–ҙлӮҳм•ј н•©лӢҲлӢӨ вҖ” write м§Ғнӣ„ н•ң cycle м•Ҳм—җ лӘЁл“  master IF к°Җ мғҲ NS лҘј л°ҳмҳҒн•ҳлҠ”м§Җ SVA лЎң к°•м ңн•ҳм„ёмҡ”.
    - EL3 лҜёкө¬нҳ„ SoC м—җм„ңлҠ” м—¬кё° лӘЁл“Ҳмқҳ лӘЁл“  "EL3" мһҗлҰ¬лҘј EL2 лҳҗлҠ” monitor-less лӘЁлҚёлЎң мһ¬н•ҙм„қн•ҙм•ј н•©лӢҲлӢӨ.
