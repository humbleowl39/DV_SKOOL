용어집 편집 시 ISO 11179 규칙을 준수한다. `**Definition.**`은 단문으로 고정하며 내용을 부풀리지 않는다. `**Source.**` 구조도 불변. Example 필드가 없거나 Related 가 빈약한 항목에만 경량 보강(예시 한 줄, See also 추가)한다.

**Why:** Definition을 길게 만들면 ISO 11179 단문 원칙 위반이고, embedded example이 생기면 정의와 예시의 경계가 모호해진다.

**How to apply:** 보강 대상 항목 판별 기준 — Example 필드 없음 OR See also 없음 OR Related가 1개 이하. 보강 내용은 기존 사실 기반 산문화만 허용, 새 신호명·수치 생성 금지.
