# Hardware Interview 퀴즈

각 챕터의 핵심 개념 이해도를 확인합니다. Bloom's Taxonomy 분포를 적용해 **암기 / 이해 / 적용 / 분석 / 평가** 수준의 문항이 섞여 있습니다.

## 챕터별 퀴즈

- **Unit 1** — [Digital Design / RTL](01_digital_rtl_quiz.md)
- **Unit 2** — [Design Verification](02_design_verification_quiz.md)
- **Unit 3** — [Embedded / Firmware](03_embedded_firmware_quiz.md)
- **Unit 4** — [Computer Architecture](04_computer_architecture_quiz.md)
- **Unit 5** — [Analog / Mixed-Signal](05_analog_mixed_signal_quiz.md)
- **Unit 6** — [Physical Design](06_physical_design_quiz.md)

## 사용법

1. 챕터 본문을 학습한 후 해당 모듈의 퀴즈 페이지로 이동
2. 정답을 머릿속으로 정한 뒤 **"정답 / 해설"** 영역을 펼쳐 확인
3. 틀린 문항은 본문 해당 절을 다시 학습
4. 인터뷰 시뮬레이션 — 답을 *말로* 30초~1분 안에 설명할 수 있어야 함
