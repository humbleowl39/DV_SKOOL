# 퀴즈 인덱스

챕터를 읽는 것과 실제로 이해한 것은 다릅니다. 이 퀴즈들은 여러분이 개념을 정말 내재화했는지 — 단순히 단어를 기억하는 수준을 넘어 계산하고, 적용하고, 틀린 주장을 판별할 수 있는지 — 확인하기 위해 설계되었습니다.

각 챕터마다 8문항 퀴즈가 있습니다. Bloom's Taxonomy로 분류:

- (Remember) / (Understand) — 정의·개념 확인
- (Apply) — 식 적용·코드 작성
- (Analyze) / (Evaluate) — 시나리오 분해·결정

기본 분포: 3 R/U + 3 Apply + 2 Analyze/Evaluate. 답·해설은 각 퀴즈 마지막 섹션 참조.
