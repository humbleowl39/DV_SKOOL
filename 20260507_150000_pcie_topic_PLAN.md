## Module 구성 (8 + Quick Ref)

1. **M01 — PCIe 동기와 진화** : PCI parallel → PCIe serial point-to-point, Gen1~Gen7 진화
2. **M02 — 3-Layer Architecture** : Transaction / Data Link / Physical 레이어와 책임
3. **M03 — TLP (Transaction Layer Packet)** : header format, MRd/MWr/CfgRd/CfgWr/Cpl, address/ID routing, traffic class
4. **M04 — DLLP & Flow Control & ACK/NAK** : ACK/NAK protocol, credit-based FC, replay buffer
5. **M05 — Physical Layer & LTSSM** : 8b/10b → 128b/130b, equalization, LTSSM state machine, lane bifurcation
6. **M06 — Configuration Space & Enumeration** : Type 0/1 header, BAR sizing, capability list, BDF, enumeration order
7. **M07 — Power Management, AER, Hot Plug** : D-state / L-state / ASPM, AER, hot plug events
8. **M08 — Advanced: SR-IOV, ATS, P2P, CXL** : VF, ATS+IOMMU, P2P, CXL.io/cache/mem 의 PCIe 와의 관계
9. **M09 — Quick Reference Card**
