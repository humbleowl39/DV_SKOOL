각 챕터의 핵심 개념 이해도를 확인합니다. Bloom's Taxonomy 분포를 적용해 **암기 / 이해 / 적용 / 분석 / 평가** 수준의 문항이 섞여 있습니다. CPU DV 면접은 정의 암기로는 첫 꼬리질문에서 막히므로, 단순 정답 맞히기보다 해설의 *왜*까지 — 그리고 가능하면 *영어로도* — 소리 내어 설명할 수 있는지를 기준으로 삼으면 어떤 변형 질문에도 대응할 수 있습니다.

## 사용법

1. 챕터 본문을 학습한 후 해당 모듈의 퀴즈 페이지로 이동
2. 정답을 머릿속으로 정한 뒤 **"정답 / 해설"** 영역을 펼쳐 확인
3. 틀린 문항은 본문 해당 절을 다시 학습
4. 인터뷰 시뮬레이션 — 답을 *말로* 30초~1분 안에 설명하고, 핵심 문항은 *영어로도* 답해 본다
