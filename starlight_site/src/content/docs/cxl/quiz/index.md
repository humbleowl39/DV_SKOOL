읽고 넘어가는 것과 설명할 수 있는 것은 다릅니다. 각 챕터의 퀴즈는 "알고 있다"고 느끼는 것과 실제 이해의 간격을 드러내기 위해 설계되었습니다. Bloom's Taxonomy 분포를 적용해 **암기/이해/적용/분석/평가** 수준의 문항이 섞여 있습니다.

## 사용법

1. 챕터 본문을 학습한 후 해당 모듈의 퀴즈 페이지로 이동
2. 정답을 머릿속으로 정한 뒤 **"정답 / 해설"** 영역을 펼쳐 확인
3. 틀린 문항은 본문 해당 절을 다시 학습
