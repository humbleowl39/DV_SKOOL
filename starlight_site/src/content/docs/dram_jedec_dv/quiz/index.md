## Bloom 레벨 표기

각 문항 끝에 `(Remember)`, `(Understand)`, `(Apply)`, `(Analyze)`, `(Evaluate)` 표시.

- **(Remember)** — 사실 회상
- **(Understand)** — 개념 설명
- **(Apply)** — 새로운 상황에 적용
- **(Analyze)** — 구성 요소 분해, 인과 관계
- **(Evaluate)** — 판단·결정·정당화

## 사용 권장

1. 챕터 본문을 읽은 후 퀴즈로 자가 점검
2. 객관식은 *answer*를 펼치기 전에 풀어볼 것
3. 대표 문제는 *사고 과정*을 따라가며 학습
