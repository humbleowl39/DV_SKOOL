## Q1. (Remember)

비트(bit)와 바이트(byte)의 관계로 옳은 것은?

- [ ] A. 1 비트 = 8 바이트
- [ ] B. 1 바이트 = 8 비트
- [ ] C. 1 비트 = 1 바이트
- [ ] D. 1 바이트 = 16 비트

<details>
<summary>정답 / 해설</summary>

**B**. 비트는 0 또는 1 한 자리(스위치 켜짐/꺼짐)이고, 바이트는 비트 8개를 묶은 단위입니다(2⁸ = 256가지 표현). 메모리 용량(KB/MB/GB)과 데이터 크기를 셀 때 바이트가 기준이 됩니다.

</details>
## Q2. (Apply)

이진수 `1010` 은 십진수로 얼마인가?

- [ ] A. 8
- [ ] B. 10
- [ ] C. 12
- [ ] D. 101

<details>
<summary>정답 / 해설</summary>

**B**. 각 자리의 무게는 오른쪽부터 1, 2, 4, 8 입니다. `1010` = 1×8 + 0×4 + 1×2 + 0×1 = 8 + 0 + 2 + 0 = **10**. 비트가 1인 자리(8과 2)의 무게만 더하면 됩니다.

</details>
## Q3. (Understand)

게이트로 계산이 다 되는데도 클럭(박자)이 필요한 이유는?

<details>
<summary>정답 / 해설</summary>

게이트를 통과하는 전기에는 _시간_ 이 걸려, 계산 결과가 안정되기 _전_ 에 다음 단계가 값을 읽으면 아직 변하는 중인 엉뚱한 값을 읽습니다. 클럭은 "한 박자(사이클) 동안 모든 계산이 안정될 시간을 준 뒤, 박자가 올 때 일제히 결과를 레지스터에 저장"하게 만들어 이 문제를 막습니다(동기 회로). 그래서 한 사이클은 "가장 긴 계산 경로가 안정될 시간"보다 길어야 하며, 이것이 클럭 속도의 상한을 정합니다.

</details>
