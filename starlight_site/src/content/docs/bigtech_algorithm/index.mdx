빅테크 코딩 면접은 알고리즘 문제를 45분 안에 푸는 형식이다. 이 과정에서 가장 흔한 실패 원인은 "올바른 코드"가 없어서가 아니라 "올바른 접근법을 제때 선택하지 못해서"다. O(N²) 풀이로 시작해 시간이 부족해지거나, 패턴을 몰라서 30분 동안 막혀 있는 상황이 여기에 해당한다. 이 코스는 Big-O 분석으로 복잡도 목표를 정하고, 입력 형태를 보고 알고리즘 패턴 후보를 빠르게 좁히는 능력을 기르는 데 집중한다. 패턴을 모르면 새 문제를 만날 때마다 처음부터 다시 생각해야 하지만, 패턴을 알면 "이 구조엔 이 기법"이라는 직관이 생겨 제한 시간 내에 완성도 있는 풀이를 보여줄 수 있다.

## 📋 사전 지식
프로그래밍 일반, 자료구조 기초
