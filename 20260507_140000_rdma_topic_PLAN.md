## Module 구성 (8 + Quick Ref)
1. **M01 — RDMA 동기와 핵심 모델** : Kernel bypass, zero-copy, Verbs API, IB↔RoCE 계보
2. **M02 — InfiniBand 프로토콜 스택** : Layer 모델, LRH/GRH/BTH/Payload/ICRC/VCRC, VL/SL
3. **M03 — RoCEv2: Ethernet 위의 RDMA** : Eth/IPv4/UDP(4791)/BTH 매핑, IB ↔ RoCEv2 차이
4. **M04 — Transport Service Types & QP** : RC/UC/UD/XRC, QP FSM (Reset → Init → RTR → RTS → SQE/SQD/Err)
5. **M05 — Memory Model: PD, MR, L_Key/R_Key, IOVA** : Memory Registration, MMU/PTW/TLB 역할
6. **M06 — Data Path: SEND/WRITE/READ/ATOMIC** : Opcode, PSN, ACK/NAK, AETH, Retry timer
7. **M07 — Congestion Control & Error Handling** : PFC/ECN/DCQCN, Local ACK timeout, RNR, R-key, QP recovery
8. **M08 — RDMA-TB 검증 환경 & DV 전략** : vrdmatb 구조, agent/env 계층, vplan, scoreboard, coverage
9. **M09 — Quick Reference Card** : opcode/header/state/error 한 장 요약
