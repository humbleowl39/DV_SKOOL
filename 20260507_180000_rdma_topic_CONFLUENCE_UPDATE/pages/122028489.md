## What is SRQ?

Shared Receive Queue is a shared resource that can be used by multiple QPs. For example, QP0,1,2 can `register` an SRQ as their Receive Queue (RQ) during create_qp(…., SRQ \* X).

##  The Inefficiency of Private Receive Queues

Assigning a private RQ to each QP is inefficient. This approach forces an application to either:

1.  Post enough receive buffers (Work Requests) on every single QP to handle its highest possible incoming rate.

2.  Rely on flow control (like `rnr_retry`), which forces the sender to back off and adds unnecessary latency when buffers are exhausted.

## How Memory is Reduced through SRQ

The Shared Receive Queue (SRQ) solves this problem by allowing multiple QPs to share a single pool of Receive Work Requests and their associated data buffers. Instead of dedicating a large number of buffers to each QP that may sit idle, the SRQ centralizes these resources. This significantly reduces wasted memory, as buffers only need to be provisioned for the aggregate traffic, not the peak rate of every individual connection.

## SRQ Operation and Management

**Buffer Sizing:**  
To prevent errors, all buffers posted to an SRQ should be sized to hold the maximum expected payload. Because WQEs are processed in strict FIFO order, an incoming message will fail if the buffer at the head of the queue is too small, even if subsequent buffers are large enough.

**The SRQ Limit Mechanism:**  
To manage the shared pool, a consumer sets an **SRQ Limit**, a threshold for the minimum number of available WQEs.

- When the WQE count drops below this limit, an asynchronous event notifies the consumer to post more buffers.

- The limit is then automatically dropped to zero to prevent repeated notifications while the consumer replenishes the queue.

- Once sufficient WQEs have been posted, the consumer must set the limit again.

**SRQ States:**  
An SRQ operates in a `non-error` or `error` state. QPs can only retrieve WQEs while the SRQ is in the `non-error` state. The only way to recover from an `error` state is to destroy the SRQ.
