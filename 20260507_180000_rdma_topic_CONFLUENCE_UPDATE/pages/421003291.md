admin@sonic:~\$ show runningconfiguration bgp Building configuration... Current configuration: ! frr version 8.5.1 frr defaults traditional hostname sonic service integrated-vtysh-config ! router bgp 65000 bgp router-id 198.18.0.1 neighbor 172.31.240.1 remote-as 65101 neighbor 172.31.240.5 remote-as 65102 ! address-family ipv4 unicast neighbor 172.31.240.1 soft-reconfiguration inbound neighbor 172.31.240.1 route-map PERMIT-ALL in neighbor 172.31.240.1 route-map PERMIT-ALL out neighbor 172.31.240.5 soft-reconfiguration inbound neighbor 172.31.240.5 route-map PERMIT-ALL in neighbor 172.31.240.5 route-map PERMIT-ALL out maximum-paths 64 exit-address-family exit ! route-map PERMIT-ALL permit 10 exit ! route-map RM_SET_SRC permit 10 exit ! end

sudo config loopback add Loopback0 sudo config interface ip add Loopback0 198.18.0.1/32 \# Leaf 1과 연결된 포트 (Port6/Ethernet 40) sudo config interface startup Ethernet40 sudo config interface mtu Ethernet40 9100 sudo config interface ip add Ethernet40 172.31.240.0/31 \# Leaf 2와 연결된 포트 (Port8/Ethernet 56/1) sudo config interface startup Ethernet56 sudo config interface mtu Ethernet56 9100 sudo config interface ip add Ethernet56 172.31.240.4/31 \# ===== 2. FRR BGP 설정 ===== sudo vtysh conf t route-map PERMIT-ALL permit 10 exit ! router bgp 65000 bgp router-id 198.18.0.1 neighbor 172.31.240.1 remote-as 65101 ! Leaf 1 neighbor 172.31.240.5 remote-as 65102 ! Leaf 2 ! address-family ipv4 unicast ! --- enable neighbor neighbor 172.31.240.1 activate neighbor 172.31.240.1 route-map PERMIT-ALL in neighbor 172.31.240.1 route-map PERMIT-ALL out neighbor 172.31.240.5 activate neighbor 172.31.240.5 route-map PERMIT-ALL in neighbor 172.31.240.5 route-map PERMIT-ALL out ! network 198.18.0.1/32 maximum-paths 64 exit-address-family end write sudo config save -y
