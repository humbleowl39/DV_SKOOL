## About

------------------------------------------------------------------------

RDMA (Remote Direct Memory Access), is a technology that enables the fast data transfer between memories (which resides in different node). Below figure descirbes the basic concept of RDMA.

<div>

</div>

It is known that RDMA technology has at least two advantages compared to traditional network stack (e.g., TCP), as below.

- Performance

  - RDMA enables ultra-fast data transfer, as it eliminates any data copies (= zero copy) in network stack (= network layer processing is entirely offloaded into the specialized hardware, so called RNIC).

  - RDMA achieves line-rate bandwidth.

- CPU utilization

  - As RDMA bypasses the network stack in both client (= the node which initiates the RDMA transfer) / server side, the CPU utilization is substantially decreased. Note that RDMA **does not require any CPU involvement at the server side, as the server-side RNIC will directly read/write the server’s memory region.**
