## How to run VMG

<div>

<div>

\# get RDMA-TB submodule git submodule update --init \# check RDMA-TB branch cd RDMA-TB git checkout {tb-branch} cd .. \# generate IPs (ippack) \# make VMG project make vcs && make export_sim_vmg \# set environment cd scripts/vmg_script unset MVP_HOME source set_env_vmg.sh \# generate DUT (only generation, no compile) cd vmg/vcs gen_dut_f_rdma cd ../../RDMA-TB/work/rdma_pcie/vsim \# if links of files are wrong ./do_if_rtl_changed.csh \# run specific test (compile + elaborate + run) mrun run --test_name vrdmatb_rdma_burst_sanity_test \# XXX After Vendor-Agnostic XXX mrun run --test_name vrdmatb_rdma_burst_sanity_test --sys_name rdma_va \# run test only (no compile, elaborate) mrun test --test_name vrdmatb_rdma_burst_sanity_test \# run regression test mrun regr --test_suite sanity_check mrun regr --test_suite network_regression \# XXX After Vender-Agnostic XXX mrun regr --test_suite va_sanity_check mrun regr --test_suite va_network_regression mrun gui
