Below is the list of RDMA opcodes supported currently.

- RC transport type

  - RDMA send

    - <div>

      SEND_FIRST, SEND_MIDDLE, SEND_LAST, SEND_ONLY

      </div>

  - RDMA send with invalidate

    - SEND_LAST_WITH_INVALIDATE, SEND_ONLY_WITH_INVALIDATE

  - RDMA send with immediate

    - SEND_LAST_WITH_IMMEDIATE, SEND_ONLY_WITH_IMMEDIATE

  - RDMA write

    - WRITE_FIRST, WRITE_MIDDLE, WRITE_LAST, WRITE_ONLY

  - RDMA write with immediate

    - WRITE_LAST_WITH_IMMEDIATE, WRITE_ONLY_WITH_IMMEDIATE

  - RDMA read

    - READ_REQUEST

  - RDMA Acknowledge

    - ACKNOWLEDGE

  - RDMA read response

    - READ_RESPONSE_FIRST, READ_RESPONSE_MIDDLE, READ_RESPONSE_LAST, READ_RESPONSE_ONLY

- UD transport type

  - RDMA send only

    - RDMA_SEND_ONLY

  - RDMA send only with immediate

    - RDMA_SEND_ONLY_WITH_IMMEDIATE
